tc-hackathon-2015
=================

## depends

- node >= v5.0.0
- npm >= v3.3.6

## installation

```
npm install
```

## start

```
npm start
```

## development

お気に入りのエディターで以下のモジュールを使えると便利

- [EditorConfig](http://editorconfig.org/): コードスタイルを揃えます
- [ESLint](http://eslint.org/): [ES2015](https://babeljs.io/docs/learn-es2015/)スタイルのjavascriptをlintします
